# ClipMain — Full-Stack экосистема для магазина одежды

Полный цикл разработки: от проектирования базы данных до мобильного приложения и админ-панели.  
Пет-проект / дипломная работа.

---

## Технологический стек

| Слой         | Технологии                                               |
| ------------ | -------------------------------------------------------- |
| **Android**  | Java, Retrofit, Glide, ViewPager2, Coroutines            |
| **Сервер**   | Java 17, Spring Boot 3, Hibernate, PostgreSQL            |
| **WPF**      | C#, .NET Framework 4.8, Entity Framework, MaterialDesign |
| **API**      | REST, DaData (подсказки адресов)                         |

---

## Возможности

### Android-приложение (Клиент)
- **Аутентификация:** Вход по номеру телефона с SMS-кодом.
- **Каталог:** Поиск с debounce, карточки товаров, листание фото (ViewPager2).
- **Корзина:** Чистая архитектура (интерфейс `CartStorage`), сохранение между сессиями.
- **Заказы:** Оформление с подсказками адресов, автозаполнение профиля, история с отменой.

### WPF Админ-панель
- **Товары:** CRUD, загрузка фото, управление размерами.
- **Пользователи:** Назначение прав администратора.
- **Заказы:** Смена статуса, просмотр состава и причин отмены.

### Сервер (Spring Boot)
- REST API (20+ эндпоинтов).
- Ролевая модель (Admin/User).
- Защита API админ-панели секретным ключом.
